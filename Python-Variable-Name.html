Inspect library’s often-used functions:

FunctionIntroduction
getmembers(obj[, predicate])Get a (k,v) tuple list of which the elements is object’s memberlist. And predicate can be used to filter the result like only getting the func object
getmodule(obj)Get the module where the object is
getfile(obj)Get the source file name where the object is
getsource(obj)Get the source code of object
isclass/isfunctino/ismethodjudge the object’s type
currentframe()Get the current stack frame object
stack([context])Get all the stack info by returning the tuple list of (frame, filename, line_number, function_name, code_context, index)
