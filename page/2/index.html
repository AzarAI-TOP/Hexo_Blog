 $2023-05-06 2023-05-23STM32 / F4073 minutes read (About 404 words)

GPIO (General Purpose Input Output)

GPIO (General Purpose Input Output) 是通用输入输出端口的简称，可以通过软件来控制其输入和输出。STM32芯片的GPIO引脚与外部设备连接起来，从而实现与外部通讯、控制以及数据采集的功能。

  • GPIO 功能描述

    • 输入
      • 输入浮空
      • 输入上拉
      • 输入下拉
      • 模拟功能
    • 输出
      • 推挽输出(具有上拉或下拉功能)
         该模式下，输出口连接到一个推挽输出的引脚。引脚在输出高电平时与电源连接，在输出低电平时与地连接。该模式下输出电流能力强，适合驱动电感、LED等负载
      • 开漏输出(具有上拉或下拉功能)
         该模式下，输出口连接到一个开漏输出的引脚。引脚在输出高电平时断开，输出低电平时与地连接。该模式下输出电流能力较弱，但适用于驱动一些需要共用信号线的设备，如I2C总线
      • 复用功能推挽(具有上拉或下拉功能)
      • 复用功能开漏(具有上拉或下拉功能)

  • GPIO 初始化步骤

    1. 确定GPIO模式：输入、通用输出、复用功能还是模拟输入(MODER)
    2. 确定GPIO输出类型：推挽输出还是开楼输出(OTYPER)
    3. 配置输出的速度(OSPEEDR)
    4. 输出的时候内部的上\下拉电阻要不要开启(PUPDR)
    5. 具体输出的内容(BSRR、ODR)

电容器在数字电路中的应用

  • 概念
